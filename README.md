# 🏋️ Gym Management System

Një aplikacion për menaxhimin e palestrës — anëtarët, trajnerët, pagesat dhe rezervimet e klasave.

## 👥 Rolet në Sistem
- **Admin** — Menaxhim i plotë
- **Staff** — Regjistrimi i pagesave dhe check-in
- **Member** — Profili personal dhe rezervimet

## 🛠️ Teknologjitë
- HTML / CSS / JavaScript
- PHP
- MySQL

## 📌 Faqet
- Dashboard
- Anëtarët
- Trajnerët
- Klasat
- Pagesat
- Check-In
